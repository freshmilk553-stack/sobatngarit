# sobatngarit
sobat Ngarit adalah proyek aplikasi yang dibuat untuk membantu, menemani, dan mempermudah aktivitas seputar pertanian dan kehidupan pedesaan. Mengusung semangat gotong royong, aplikasi ini hadir sebagai sobat bagi para petani dan pegiat agrikultur agar kerja di ladang jadi lebih efektif, modern, dan terhubung.
